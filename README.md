# Fatigue Detection & Safety Dashboard

Web dashboard untuk analisa fatigue & distraction detection. Ditujukan untuk operasional tambang / fleet.

## Fitur
- KPI: total alerts, unique operator, avg duration
- Trend daily & hourly, shift distribution
- Top operators & assets
- QC/Validation summary
- Export filtered CSV
- Auto generated insights (text)
- Sidebar filter: date, operator, fleet, shift
